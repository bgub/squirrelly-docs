## Syntax

```js
Sqrl.defineFilter('filtername', function(str) {
  // Do something with string
  return str
})
```

## Example

```js
Sqrl.defineFilter('reverse', function(str) {
  var out = ''
  for (var i = str.length - 1; i >= 0; i--) {
    out += String(str).charAt(i)
  }
  return out || str
})
```

Then in your template:

```sqrl
{{somevalue | reverse}}
```
