Squirrelly tries to follow best practices, and provides a UMD build to support most JS loading options, like ES modules, CommonJS, and AMD.

## Install Squirrelly

  <TabList>
    <Tab tabFor="Node.js">Node.js</Tab>

    <br/>

```sh
npm install squirrelly --save
```

Or if you prefer Yarn:

```sh
yarn add squirrelly
```

<br />

### Importing / Requiring

Sqrl is packaged as a UMD module, so you can require with CommonJS, import using ES Modules, or use AMD.

```js
import * as Sqrl from 'squirrelly'
// or
var Sqrl = require('squirrelly')
```

### Unpkg

```html
<script src="https://unpkg.com/squirrelly@7/dist/squirrelly.min.js"></script>
```

### JSDelivr

```html
<script src="https://cdn.jsdelivr.net/npm/squirrelly@7/dist/squirrelly.min.js"></script>
```

This makes Squirrelly available through the global `Sqrl` variable.
